# Reel Brief Template

Copy everything below, fill it in, and paste it into Claude Code.

```
=== REEL BRIEF ===

CONCEPT: [One sentence — what's this reel about?]

VIBE: [Funny / urgent / informative / aspirational / FOMO / debate-starter]

CLIPS (in order they should appear):
1. [filename] — [What this shows. Be specific.]
2. [filename] — [What this shows]
3. [filename] — [What this shows]
(add more as needed)

CAPTION SCRIPT (what the viewer reads — in order):
- [First line — this is the HOOK]
- [Second line]
- [Third line]
- [Last line — the PUNCHLINE or CTA]

TEXT OVERLAYS (big punch text that pops up, optional):
- "[text]" appears during clip [X]

FACECAM: [yes/no]

BACKGROUND MUSIC: [filename or "none"]

CTA: [e.g. "Download Nitely — link in bio"]

NOTES: [Speed up clip 2 / make it urgent / this is for the flash challenge series / etc.]

=== END BRIEF ===
```
