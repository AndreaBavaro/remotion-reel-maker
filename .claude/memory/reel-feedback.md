# Reel Production Feedback Log

This file is the **learning brain** for the reel-planner and reel-editor skills. Every time a reel is completed, the user rates it and gives feedback. That feedback is logged here as structured learnings.

**Both skills MUST read this file before starting any work.** The learnings here override default behavior when they conflict — this is how the system improves over time.

---

## How to read this file

Each entry has:
- **Reel ID** — which reel the feedback is about
- **Rating** — 1-10 (1 = terrible, 10 = perfect)
- **Keep** — things that worked well and should be repeated
- **Change** — things that should be done differently next time
- **Learning** — the extracted rule that future reels should follow

Learnings are cumulative. A learning from reel-003 applies to reel-004, reel-005, and every reel after it unless a later learning contradicts it (in which case the newer learning wins).
